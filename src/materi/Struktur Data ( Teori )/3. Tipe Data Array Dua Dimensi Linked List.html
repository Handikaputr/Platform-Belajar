Space-Efficient Linked List menggabungkan keunggulan array (cache-friendly) dengan linked list (fleksibel). Setiap node menyimpan beberapa elemen sekaligus.

Struktur Space-Efficient LL

Terdiri dari node-node yang masing-masing menyimpan array kecil. Berguna untuk menghemat overhead pointer.

Keunggulan & Trade-off
Keunggulan
Lebih cache-friendly dari DLL biasa. Overhead pointer lebih sedikit. Akses elemen dalam node = O(1).
Trade-off
Insert/delete memerlukan pergeseran elemen dalam node. Implementasi lebih kompleks dari SLL/DLL biasa.
