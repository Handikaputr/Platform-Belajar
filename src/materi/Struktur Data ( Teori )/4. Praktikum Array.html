1. Basic Java App

Untuk memulai, kita perlu struktur dasar public class dan main method. Klik teks di bawah untuk melihat anatomi kodenya:

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello World!");
  }
}

2. Variables & Types

Cara Deklarasi:

type-name variable-name;

  • int : Angka bulat (10, 20)
  • double : Angka pecahan (10.5)
  • String : Teks ("Halo")
  • boolean : True/False

4. Input Keyboard (Scanner)

Untuk mengambil input, kita butuh class Scanner dari library java.util.

Langkah-langkah:

  1. Import Scanner
  2. Buat object Scanner
  3. Gunakan nextLine() atau nextInt()
import java.util.Scanner;

Scanner input = new Scanner(System.in);
String nama = input.nextLine();
