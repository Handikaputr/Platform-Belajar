Mencetak elemen array — bisa per indeks tertentu atau semua elemen dengan perulangan for loop.

Program: Mencetak Array 1D
arrayCetak1.java
public class arrayCetak1 {
  public static void main(String[] args) {

    String[] cetakA = new String[5];

    // Isi nilai ke setiap indeks
    cetakA[0] = "Australia";
    cetakA[1] = "Ceko";
    cetakA[2] = "Jepang";
    cetakA[3] = "Indonesia";
    cetakA[4] = "Kanada";

    // Cetak elemen ke-2 (indeks 1)
    System.out.println("Negara ke-2: " + cetakA[1]);

    // Cetak semua elemen dengan for loop
    for (int i = 0; i < 5; i++) {
        System.out.println(cetakA[i]);
    }
  }
}
OUTPUT:
Negara ke-2: Ceko
Australia
Ceko
Jepang
Indonesia
Kanada
STEP 01 — INISIASI
String[] cetakA = new String[5];
Buat array kosong berkapasitas 5 elemen bertipe String.
STEP 02 — ISI NILAI
cetakA[0] = "Australia";
Isi setiap slot dengan nilai. Indeks dimulai dari 0, bukan 1.
STEP 03 — AKSES SPESIFIK
cetakA[1] → "Ceko"
Akses langsung via indeks. Hasilnya nilai pada posisi tersebut.
STEP 04 — FOR LOOP
for (int i=0; i<5; i++)
Perulangan dari indeks 0 sampai 4 (5 iterasi total) untuk mencetak semua elemen.

Mengambil input dari keyboard menggunakan class Scanner. Diperlukan import java.util.Scanner; di awal file.

Input Array dari Keyboard — Latihan 1
negaraA.java
import java.util.Scanner; // import class Scanner untuk input keyboard

public class negaraA {
  public static void main(String[] args) {

    Scanner input = new Scanner(System.in); // buat Scanner untuk input
    String[] negara = new String[5]; // array 5 elemen String

    System.out.println("Silahkan masukkan nama negara!");

    for (int i = 0; i < 5; i++) {
        System.out.print("Masukkan negara ke[" + (i+1) + "]:");
        negara[i] = input.nextLine(); // baca satu baris teks dari keyboard
    }

  }
}
SIMULASI I/O:
Silahkan masukkan nama negara!
Masukkan negara ke[1]: Australia
Masukkan negara ke[2]: Ceko
Masukkan negara ke[3]: Jepang
Masukkan negara ke[4]: Indonesia
Masukkan negara ke[5]: Kanada
Penjelasan Scanner
import
Penanda untuk menggunakan class dari library Java. Scanner ada di java.util.
Scanner input
Membuat objek Scanner. System.in = keyboard sebagai sumber input.
nextLine()
Membaca satu baris teks dari keyboard sampai user tekan Enter.
nextInt()
Membaca input integer dari keyboard. Ada juga nextDouble(), nextFloat(), dll.

ArrayList<String> list = new ArrayList<>();
