Dalam Java, identifiers terdiri dari classes, variables, dan subroutines. Variable adalah nama yang digunakan untuk menunjuk data di memori.

Konsep Variable
Dunia Nyata
variabel = bilangan
tipe data = ganjil
Di Komputer (Java)
int bil = 7;
// bil = variabel

Programmer hanya perlu mengingat nama variable — tidak perlu mengingat alamat memori fisik tempat data disimpan.

Deklarasi di Java
// Deklarasi array kosong
1int[] A = new int[5]; // 5 elemen, nilai default 0
2 
// Deklarasi array berisi nilai
3String[] negara = {"Australia", "Ceko", "Jepang"};

7for (int i=0; i<A.length; i++) {
8    System.out.println(A[i]);
9}

Setiap elemen array tersimpan di alamat memori yang berurutan (contiguous). Penomoran alamat bergantung pada komputer yang digunakan.

Alamat array bersebelahan dengan indeks array — setiap elemen menempati blok memori yang berdekatan.
Penomoran alamat bergantung komputer. Komputer berbeda bisa memiliki alamat berbeda untuk nama variable yang sama.

ArrayList adalah array dinamis di Java — ukurannya bisa berubah saat runtime, berbeda dengan array biasa yang ukurannya tetap.

Deklarasi ArrayList
ArrayList<String> namelist;
Deklarasi variabel ArrayList bertipe String
namelist = new ArrayList<String>();
Inisiasi objek ArrayList dengan operator new
ArrayList<Player> playerList = new ArrayList<Player>();
Satu baris: deklarasi + inisiasi sekaligus
Perbandingan Array vs ArrayList
Aspek Array ArrayList
UkuranTetap (fixed)Dinamis (otomatis)
Tipe dataPrimitif & ObjectObject (wrapper class)
Deklarasiint[] A = new int[5]ArrayList<Integer> A
AksesA[i]A.get(i)
