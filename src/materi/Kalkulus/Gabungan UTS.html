📐 Konsep Dasar

Dari "suku itu apa" sampai "kenapa integral adalah kebalikan turunan". Baca pelan-pelan.

✦ Suku itu Apa?

Dalam aljabar, suku adalah tiap bagian terpisah dalam sebuah fungsi, dipisahkan oleh + atau −.

f(x) = 3x² − 4x + 7

Fungsi ini punya 3 suku: 3x², -4x, dan 7

Bagian-bagian suku 3x² → angka 3 disebut koefisien, x adalah variabel, 2 adalah eksponen/pangkat
↗ Turunan (Derivatif) itu Apa?
Bayangkan kamu naik motor. Posisi kamu di jalan itu fungsi F(x). Turunan adalah seberapa cepat posisi itu berubah — alias kecepatan. Kalau posisinya F(x) = x², maka kecepatan (turunan) = 2x.

Aturan turunan yang perlu kamu ingat:

Fungsi f(x)Turunan f'(x)Contoh
xⁿn·xⁿ⁻¹x³ → 3x²
c (konstan)07 → 0
eˣeˣeˣ → eˣ (tetap!)
sin xcos xsin x → cos x
cos x-sin xcos x → -sin x
ln x1/xln x → 1/x
Cara kerja aturan pangkat Turunkan xⁿ: kalikan pangkat ke depan, kurangi pangkat 1.
xⁿ → n · xⁿ⁻¹  |  Contoh: x⁵ → 5x⁴  |  3x² → 3·2x¹ = 6x
↙ Integral = Anti-Turunan
Kalau turunan itu seperti "maju" (dari fungsi ke kecepatan), maka integral itu "mundur" (dari kecepatan kembali ke fungsi asalnya). Integral adalah operasi kebalikan turunan.

Hubungannya:

F'(x) = f(x) ←→ ∫ f(x) dx = F(x) + C

Artinya: kalau F diturunkan menghasilkan f, maka integral dari f adalah F.

Kenapa ada + C? Karena ada banyak fungsi yang kalau diturunkan menghasilkan hal yang sama. Contoh: x² + 5, x² + 100, x² − 3 semuanya kalau diturunkan jadi 2x. Jadi kita tulis C (konstanta sembarang) untuk mewakili semua kemungkinan itu.
Cara baca notasi ∫ 6x² dx dibaca: "integral dari 6x² terhadap x"
Simbol ∫ = tanda integral (huruf S besar, dari kata "summa")
dx = "terhadap x" (dari kata "differentia")
📐 Refresher: Sin, Cos, Tan

Yang kamu perlu ingat untuk integral, bukan segitiga — tapi rumus integral & turunannya:

∫ cos x dx = sin x + C (positif)

✏️ Integral Tak Tentu

Integral tanpa batas atas-bawah. Hasilnya selalu ada +C. Ini yang paling banyak keluar di kisi-kisi.

📜 6 Aturan Wajib Hafal
1. Aturan Pangkat
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C
asal n ≠ -1
2. Kasus Khusus n=-1
∫ x⁻¹ dx = ∫(1/x)dx = ln|x| + C
3. Konstanta
∫ c·f(x)dx = c·∫f(x)dx
konstanta bisa keluar
4. Penjumlahan
∫[f±g]dx = ∫f dx ± ∫g dx
integralkan suku per suku
5. Eksponensial
∫ eˣ dx = eˣ + C
eˣ adalah keajaiban matematika!
6. Nol
∫ 0 dx = C
integral nol = konstanta
✦ Cara Pakai Aturan Pangkat — Step by Step
Kalau turunan itu "kalikan pangkat, kurangi 1", maka integral kebalikannya: "tambah 1 ke pangkat, bagi dengan pangkat baru".
∫ xⁿ dx = x^(n+1) / (n+1) + C
  • 1
    Lihat pangkat n
  • 2
    Tambah 1 ke pangkat: n → n+1
  • 3
    Bagi koefisien dengan n+1
  • 4
    Tambahkan +C di akhir
Contoh A — Soal UTS No.1Pangkat biasa
∫ 6x² dx
Pangkat n = 2. Konstanta = 6. Pakai rumus pangkat.
Hitung = 6 · x^(2+1) / (2+1) = 6 · x³/3
Sederh. = 6/3 · x³ = 2x³
= 2x³ + C
Verifikasi balik Turunkan 2x³ + C → 2·3x² = 6x² ✓ Sama dengan soal awal!
✦ Integralkan Suku Per Suku

Kalau ada penjumlahan/pengurangan, integralkan masing-masing suku secara terpisah.

Contoh B — Soal UTS No.23 suku berbeda
∫ (3x² − 4x + 7) dx
Pisah = ∫3x² dx − ∫4x dx + ∫7 dx
Suku 1 ∫3x² dx = 3 · x³/3 = x³
Suku 2 ∫4x dx = 4 · x²/2 = 2x²
Suku 3 ∫7 dx = 7x    (konstanta × x)
= x³ − 2x² + 7x + C
Contoh C — Soal UTS No.3eˣ + trigonometri
∫ (2eˣ − 3sin x + 4cos x) dx
eˣ ∫ 2eˣ dx = 2eˣ
sin x ∫ −3sin x dx = −3 · (−cos x) = +3cos x
cos x ∫ 4cos x dx = 4sin x
= 2eˣ + 3cos x + 4sin x + C
⚠️ Jebakan! ∫ sin x dx = −cos x (ada minusnya). Jadi kalau ada −3sin x, hasilnya −3 × (−cos x) = +3cos x. Minus × minus = plus!
Berapa hasil dari ∫ x⁵ dx?
🔀 Integral Substitusi Sederhana

Teknik ganti variabel: ubah bentuk susah ke bentuk yang bisa pakai aturan dasar.

💡 Ide Dasarnya
Kamu ketemu soal kayak ∫ (2x+1)⁵ dx. Kalau kamu expand itu jadi polinomial panjang banget. Tapi ada cara pintas: ganti bagian dalam (2x+1) dengan huruf u, selesaikan integral u yang lebih simple, lalu ganti balik.
Jika u = g(x), maka du = g'(x) dx
  • 1
    Pilih u — biasanya bagian dalam kurung atau yang ribet
  • 2
    Cari du — turunkan u terhadap x, tulis jadi du = ... dx
  • 3
    Ubah dx — dari du = ... dx → dx = du/...
  • 4
    Substitusi ke integral asli, selesaikan dalam u
  • 5
    Ganti balik u dengan ekspresi x aslinya
✦ Pola 1: Bentuk (ax + b)ⁿ
Contoh — Soal UTS No.4Linear dalam kurung berpangkat
∫ (2x + 1)⁵ dx
Pilih u u = 2x + 1    (bagian dalam kurung)
Cari du du/dx = 2  →  du = 2 dx  →  dx = du/2
Substitusi ∫ u⁵ · (du/2) = (1/2) ∫ u⁵ du
Selesaikan (1/2) · u⁶/6 = u⁶/12 + C
Balik Ganti u = 2x + 1
= (2x + 1)⁶ / 12 + C
✦ Pola 2: Bentuk 1/(ax + b) — Hasilnya Logaritma
Ingat! ∫ 1/u du = ln|u| + C   (ini dari aturan n = -1)
Contoh — Soal UTS No.5Pecahan → ln
∫ 3/(3x − 2) dx
Pilih u u = 3x − 2
Cari du du = 3 dx  →  dx = du/3
Substitusi ∫ 3/u · (du/3) = ∫ (1/u) du
Balik = ln|u| + C = ln|3x − 2| + C
= ln|3x − 2| + C
Kalau u = 5x + 3, berapa du?
🧮 Substitusi Fungsi Aljabar

Versi lanjut substitusi — di sini u bukan linear, tapi ada x² atau x³ di dalamnya. Kuncinya: harus ada turunan u di luar kurung.

🔍 Cara Mengenali Polanya
Kalau kamu lihat soal integral dan ada dua bagian yang salah satunya kelihatan seperti turunan dari yang lain — itulah tandanya pakai substitusi aljabar.
Contoh pengenalan pola ∫ x(x² + 3)⁴ dx
→ Ada x² + 3 dalam kurung. Turunan x² + 3 adalah 2x.
→ Di luar kurung ada x (hampir sama dengan 2x, tinggal urus konstantanya!).
→ Cocok untuk substitusi u = x² + 3.
✦ Pola A: Ada faktor yang cocok dengan du
Contoh — Soal UTS No.6Turunan u muncul di luar
∫ x(x² + 3)⁴ dx
Kenali u = x² + 3   (bagian dalam kurung)
du du = 2x dx  →  x dx = du/2
Substitusi ∫ u⁴ · x dx = ∫ u⁴ · (du/2) = (1/2)∫ u⁴ du
Selesaikan (1/2) · u⁵/5 = u⁵/10 + C
Balik u = x² + 3
= (x² + 3)⁵ / 10 + C
✦ Pola B: Pembilang = Turunan Penyebut → Langsung ln
Pola emas yang sering muncul Kalau integral berbentuk ∫ f'(x)/f(x) dx, jawabannya langsung:
= ln|f(x)| + C
Contoh — Soal UTS No.7Pembilang = turunan penyebut
∫ (2x + 1)/(x² + x + 4) dx
Cek Penyebut: x² + x + 4. Turunannya: 2x + 1.
Perhatikan Pembilang (2x+1) TEPAT SAMA dengan turunan penyebut!
u u = x² + x + 4  →  du = (2x+1) dx
Subs. ∫ du/u = ln|u| + C
= ln|x² + x + 4| + C
Cara cepat hafalnya ∫ (turunan penyebut)/(penyebut) dx = ln|penyebut| + C
Untuk ∫ x²(x³ + 1)⁵ dx, apa pilihan u yang tepat?
∂ Integral Parsial

Dipakai kalau integrand adalah perkalian dua fungsi berbeda jenis (misal polinom × eksponensial, atau polinom × trigonometri).

📜 Rumus Dasar
∫ u dv = uv − ∫ v du

Cara baca: "integral u kali dv sama dengan uv dikurangi integral v kali du"

Panduan pilih u dan dv — Aturan LIATE Pilih u dari yang LEBIH KIRI di daftar ini:
Logaritma → Invers trigonometri → Aljabar/polinom → Trigonometri → Eksponensial
Logikanya u itu yang kita turunkan (harusnya makin simple). dv itu yang kita integralkan. Polinom kalau diturunkan terus lama-lama jadi 0 — makanya polinom jadi u.
✦ Cara Pakai Rumus — Parsial Berulang
Contoh — Soal UTS No.8Polinom × eˣ, parsial 2x
∫ x²eˣ dx
x² itu aljabar, eˣ itu eksponensial. Berdasarkan LIATE, u = x².
Set 1 u = x², dv = eˣ dx  →  du = 2x dx, v = eˣ
Rumus = uv − ∫v du = x²eˣ − ∫eˣ · 2x dx
Sisa Masih perlu selesaikan ∫ 2xeˣ dx → parsial lagi!
Set 2 u = 2x, dv = eˣ dx  →  du = 2 dx, v = eˣ
Rumus ∫ 2xeˣ dx = 2xeˣ − ∫ 2eˣ dx = 2xeˣ − 2eˣ
Gabung x²eˣ − (2xeˣ − 2eˣ) = x²eˣ − 2xeˣ + 2eˣ
= eˣ(x² − 2x + 2) + C
📊 Metode Tabulasi — Cara Lebih Cepat
Daripada parsial berulang yang ribet, bikin tabel: kolom kiri turunkan sampai 0, kolom kanan integralkan terus. Kali diagonal dengan tanda bergantian +−+−...
Format Tabel Kolom Tanda | Kolom D (diturunkan) | Kolom I (diintegralkan)
Kalikan diagonal: D baris n × I baris n+1, dengan tanda baris n.
Contoh — Soal UTS No.9Polinom × sin x
∫ x² sin x dx
f(x) = x² (turunkan), g(x) = sin x (integralkan)
TandaD (turunan)I (integral)
+x²sin x
−2x−cos x
+2−sin x
−0cos x
Perkalian diagonal:
Baris + (+) x² × (−cos x) = −x²cos x
Baris − (−) 2x × (−sin x) = +2x sin x
Baris + (+) 2 × cos x = +2cos x
Stop D = 0 → selesai
= −x²cos x + 2x sin x + 2cos x + C
Untuk ∫ x·eˣ dx, berdasarkan LIATE mana yang jadi u?

No. 1 — A. Pengantar Dasar Integral

Jelaskan hubungan antara turunan dan integral, lalu tentukan:

∫ 6x² dx
Penjelasan hubungan Integral adalah operasi kebalikan (anti-turunan) dari turunan. Jika F'(x) = f(x), maka ∫ f(x) dx = F(x) + C. Contoh: Karena turunan x³ = 3x², maka integral 3x² = x³ + C.
Aturan ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C. Di sini n = 2, koef = 6.
Hitung = 6 · x^(2+1)/(2+1) = 6 · x³/3 = 2x³
= 2x³ + C
No. 2 — B. Integral Tak Tentu
∫ (3x² − 4x + 7) dx
∫3x²= 3·x³/3 = x³
∫−4x= −4·x²/2 = −2x²
∫7= 7x
= x³ − 2x² + 7x + C
No. 3 — B. Integral Tak Tentu
∫ (2eˣ − 3sin x + 4cos x) dx
∫2eˣ= 2eˣ
∫−3sinx= −3 × (−cos x) = +3cos x   ⚠️ minus × minus = plus!
∫4cosx= 4sin x
= 2eˣ + 3cos x + 4sin x + C
No. 4 — C. Substitusi Sederhana
∫ (2x + 1)⁵ dx
uu = 2x + 1
dudu = 2 dx  →  dx = du/2
Subs.∫ u⁵ · (du/2) = (1/2)·u⁶/6 = u⁶/12 + C
Baliku = 2x + 1
= (2x + 1)⁶ / 12 + C
No. 5 — C. Substitusi Sederhana
∫ 3/(3x − 2) dx
uu = 3x − 2
dudu = 3 dx  →  dx = du/3
Subs.∫ 3/u · (du/3) = ∫ (1/u) du = ln|u| + C
= ln|3x − 2| + C
No. 6 — D. Substitusi Fungsi Aljabar
∫ x(x² + 3)⁴ dx
Kenali pola u = x² + 3. Turunannya 2x. Di luar ada x. Cocok!
uu = x² + 3
dudu = 2x dx  →  x dx = du/2
Subs.∫ u⁴ · (du/2) = (1/2) · u⁵/5 = u⁵/10 + C
Baliku = x² + 3
= (x² + 3)⁵ / 10 + C
No. 7 — D. Substitusi Fungsi Aljabar
∫ (2x + 1)/(x² + x + 4) dx
Trik deteksi: cek apakah pembilang = turunan penyebut Penyebut: x² + x + 4. Turunannya: 2x + 1. Pembilang: 2x + 1. SAMA PERSIS! → langsung ln.
uu = x² + x + 4
dudu = (2x + 1) dx   (tepat = pembilang)
Subs.∫ du/u = ln|u| + C
= ln|x² + x + 4| + C
No. 8 — E. Integral Parsial (Rumus)
∫ x²eˣ dx
LIATE: x² = Aljabar, eˣ = Eksponensial → u = x² Perlu parsial dua kali karena derajat polinom = 2.
Iterasi 1:
Setu = x², dv = eˣ dx  →  du = 2x dx, v = eˣ
Hasil= x²eˣ − ∫ eˣ·2x dx
Iterasi 2 (selesaikan ∫2xeˣ dx):
Setu = 2x, dv = eˣ dx  →  du = 2 dx, v = eˣ
Hasil= 2xeˣ − ∫ 2eˣ dx = 2xeˣ − 2eˣ
Gabungkan:
Finalx²eˣ − (2xeˣ − 2eˣ) = x²eˣ − 2xeˣ + 2eˣ
= eˣ(x² − 2x + 2) + C
No. 9 — E. Integral Parsial (Tabulasi)
∫ x² sin x dx
f(x) = x² → turunkan sampai 0. g(x) = sin x → integralkan terus.
TandaD (turunan)I (integral)
+x²sin x
−2x−cos x
+2−sin x
−0cos x

−2x × (−sin x) = +2x sin x   (minus × minus = +)

= −x²cos x + 2x sin x + 2cos x + C

A. Rumus Dasar Integral Tak Tentu
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C  (n≠−1)
∫ x⁻¹ dx = ln|x| + C
∫ eˣ dx = eˣ + C
∫ c·f(x) dx = c·∫f(x) dx
∫ [f±g] dx = ∫f dx ± ∫g dx
∫ 0 dx = C
B. Integral Trigonometri
∫ sin x dx = −cos x + C
∫ cos x dx = sin x + C
∫ sec²x dx = tan x + C
∫ csc²x dx = −cot x + C
∫ tan x dx = ln|sec x| + C
∫ sec x dx = ln|tan x + sec x| + C
C. Substitusi — Langkah
1. Pilih u (bagian dalam / penyebut / yang ribet)
2. Cari du = turunan(u) · dx
3. Ubah: dx = du / turunan(u)
4. Substitusi semua x → u, selesaikan
5. Ganti balik u → ekspresi x
D. Pola Substitusi Aljabar
∫(ax+b)ⁿ dx → u=ax+b, du=a dx
∫1/(ax+b) dx = (1/a)·ln|ax+b|+C
∫f'(x)/f(x) dx = ln|f(x)| + C ⭐
∫x(x²+c)ⁿ dx → u=x²+c, x dx=du/2
E. Integral Parsial
Rumus: ∫ u dv = uv − ∫ v du
Pilih u: LIATE → L=log, I=inv.trig, A=aljabar, T=trig, E=eˣ
u = yang paling kiri di LIATE
Tabulasi: kolom D(turunkan) dan I(integralkan), kali diagonal +−+−
